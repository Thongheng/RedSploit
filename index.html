Quick Start Guide

1 Clone & Install
git clone https://github.com/Thongheng/RedSploit.git
cd RedSploit
./setup.sh
2 Start Shell
red
3 Set Target & Run
set target 10.10.10.1
use infra
nmap -sV -sC {target}
